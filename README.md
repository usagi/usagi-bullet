# うさぎ★ばれっと

このリポジトリーは @usagi が Bullet Physics Libraryをにゃんにゃんする文書群です。
基本的にSDKに含まれるDemoの中身を日本語にしたりC++11にしたりします。

## このリポジトリーの開発環境

- Linux Mint 15 KDE
    - g++ 4.7.3
    - cmake 2.8.10.1
    - ninja 1.0.0

### 動作確認（例: Demos/HelloWorld）

    cd Demos/HelloWorld
    mkdir tmp
    cd tmp
    cmake -G Ninja ..
    ninja
    ./AppHelloWorld
